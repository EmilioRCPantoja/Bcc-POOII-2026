## Folder Contents

- **01-basic**  
    Contains introductory examples demonstrating basic SQLAlchemy setup, model definition, and simple queries.
    It uses SQLite.

- **02-crud**  
    Includes code samples for Create, Read, Update, and Delete operations using SQLAlchemy ORM.

- **03-relation**  
    Provides examples of defining and working with relationships between tables, such as one-to-many and many-to-many associations.
    Also, contains ways to connect to some databases, like Postgres and Mysql.

- **04-inheritancve**  
    Features demonstrations of model inheritance a pattern in SQLAlchemy: the joined-table inheritance.
